## Subscribe to full version
**Use Case ID:** UC10

**Brief Description:**
UC2 allows the user to purchase a subscription in order to unlock premium features of FlowFinder.

**Primary Actors:**
User

**Secondary Actors:** -

**Preconditions:**
1. The user is currently in basic mode of the application.

**Main Flow of Events:**
1. The user selects the option to subscribe to the full version.
2. The system displays available subscription plans (monthly, yearly) with pricing details.
3. The user chooses a subscription plan.
4. The system requests confirmation of the purchase.
5. The user confirms the subscription.
6. INCLUDE (Pay for subscription)
7. The system activates the subscription and unlocks premium features.
8. The system displays a confirmation message of successful subscription.

**Alternative Flows:**

5a **User Cancels:** If the user cancels before confirming, the process ends and the application remains in basic mode.

5b **System Error:** If the subscription process fails, the system notifies the user and the application remains in basic mode.

**Post-conditions:**
- If successful, the user has access to premium features until the subscription expires.
- If unsuccessful or cancelled, the application remains in basic mode.
