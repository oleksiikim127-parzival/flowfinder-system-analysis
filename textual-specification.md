## Manage financial transactions
- id: UC2

UC2 updates, and deletes transactions to keep financial records accurate and up to date. 

## Manage categories
- id: UC4

UC4 creates new categories, renames existing categories, and deletes categories. Category can't be deleted if it is assigned to some transaction.

## Choose main currency
- id: UC5

UC5 sets or changes the main currency used for displaying balances and statistics. All amounts are converted to the selected currency for a consistent overview.

## Filter transactions
- id: UC6

UC6 filters transactions by date range, category, amount, or description to quickly find specific records.

## View basic statistics
- id: UC7

UC7 shows simple financial summaries, such as total income, expenses, and balance over a selected period or broken down by category.

# Manage transaction templates
- id: UC8

UC8 allows user to create transaction templates to simplify frequent entry of similar payments

## Import/Export data  
- id: UC9

UC9 allows user to import transaction data from an external file or export their existing data for backup, migration, or bulk editing.  

## Synchronize devices
- id: UC11

UC11 allows premium user to synchronize their data across multiple devices with offline-first support.  

## Connect bank account  
- id: UC12

UC12 allows premium user to connect FlowFinder to their bank account through supported APIs.  

## Download new bank transactions  
- id: UC13

UC13 downloads regularly new transactions from connected bank accounts.  

## Create payment order  
- id: UC14

UC14 allows a premium user to create a payment order **without** needing to use a **banking application**.  

## Show advanced statistics  
- id: UC15

UC15 allows a premium user to view advanced statistics. It displays **detailed graphs** describing the user’s budgeting.  

## Cancel a subscription  
- id: UC16

UC16 allows a premium user to **cancel their subscription**. After canceling, the application automatically downgrades to **basic mode**.  

## Manage payment reminders  
- id: UC17

UC17 allows a premium user to **create, edit, or delete reminders** for important payments.  

## Remind payment  
- id: UC18

UC18 enables the system to **send reminders** for important payments. These reminders must have been previously set by the premium user through UC18.  

## Manage budget limits  
- id: UC19

UC19 allows a user to **create, edit, or delete budget limits** for various categories.  

## Auto-download current rates  
- id: UC20

UC20 enables the system to automatically synchronize its exchange rates with external sources.

## Manage groups  
- id: UC21

UC21 allows a premium user to **create, edit, or delete user groups** for sharing budgeting information.

## Pay for subscription
- id: UC22

UC22 allows the user to pay for a subscription through a payment gateway.
